## Features

| Feature | Description |
|---------|-------------|
| **Real-time Tracking** | See intents, work items, and runs update as you work |
| **Three-Tab Interface** | Switch between Runs, Specs, and Overview views |
| **Smart Suggestions** | AI-recommended next actions based on project state |
| **Dependency Visualization** | See what's blocked and what's ready to start |
| **Activity Timeline** | Track recent completions and progress |

## Quick Start

1. **Install** from [VS Code Marketplace](https://marketplace.visualstudio.com/items?itemName=fabriqaai.specsmd) or [Open VSX](https://open-vsx.org/extension/fabriqaai/specsmd)
2. **Initialize** specsmd in your project:
   ```bash
   npx specsmd install
   ```
3. **Open** the specsmd sidebar (look for the specsmd icon in the activity bar)

---

## Sidebar Views

### Runs Tab
Track active runs and completed runs with progress indicators, stage pipelines, and work item status.

### Specs Tab
Browse your project specifications hierarchically: Intents → Units → Stories/Work Items. Click any item to open its markdown file.

### Overview Tab
High-level project metrics, overall progress percentage, and AI-suggested next actions.
